## Setup Environment

### 1. Using Pipenv (Recommended)
```sh
pipenv install
```

### 2. Using requirements.txt (Optional)
If you prefer not to use `pipenv`, you can install dependencies using `requirements.txt`:
```sh
pip install -r requirements.txt
```

## Run Streamlit App
```sh
streamlit run dashboard.py
```
